## 2. 教室の3つの特徴（HP強調ポイント）

### ① カフェで学ぶ

## 3. 日記メソッド（最大の特徴・オリジナル学習法）

この教室が最も重視する学習法。「書く→読む→話す」の3ステップで、インプットとアウトプットを同時に鍛える。

### 📝 仕組み

| ステップ | 内容 |
|----------|------|
| ① 事前準備 | 生徒が毎回3〜5文の韓国語日記を書いてくる（テーマ指定あり） |
| ② 音読 | 授業冒頭で生徒が日記を声に出して読む |
| ③ 発音矯正 | 先生がリアルタイムでフィードバック・ネイティブ表現に修正 |
| ④ 質疑応答 | 日記の内容について先生が韓国語で質問 → 生徒が韓国語で答える |
| ⑤ 語彙拡張 | 日記に出てきた表現をベースに関連語彙・フレーズを学ぶ |
| ⑥ 次回テーマ | 次回の日記テーマを提示（例：「週末にしたこと」「好きな食べ物」） |

### ✨ メリット

1. 自分の話だから興味が湧き、記憶に残りやすい
2. 毎回話す機会があるので実戦感覚が身につく
3. 完璧でなくてOK → 初心者も気軽に始められる

### 🌐 サイトでの表現案

- 「あなたの物語で学ぶ韓国語」
- 「教科書ではなく、日記から始まる会話レッスン」
- 「毎週、自分の日常を韓国語で表現しながら自然に話せるようになります」

### ■ 日記テーマ例（週替わり）

- 今週あったこと・週末の出来事
- 最近見たドラマ・聴いた曲
- 好きな食べ物・行きたいレストラン
- 仕事・学校での出来事
- 将来の夢・やりたいこと
- 最近買ったもの・ほしいもの

## 5. カリキュラム概要

レベルは入会時の体験レッスン（無料）で確認。初心者から始めた場合、約1年で実用的な韓国語会話が身につくよう設計。

| レベル | 期間目安 | テーマ・学習内容 | 到達目標 |
|--------|----------|------------------|----------|
| Level 0 | 2〜3ヶ月 | ハングル・発音・基礎フレーズ | 自己紹介・あいさつができる |
| Level 1 | 3〜4ヶ月 | 日常会話・基本文法・日記スタート | 日常的なやりとりができる |
| Level 2 | 3〜4ヶ月 | 応用会話・テーマ別語彙・日記読解 | 自分の意見を話せる |
| Level 3 | 3〜4ヶ月 | ビジネス会話・ニュース・ディスカッション | 幅広いトピックで話せる |

### Level 0 ｜ 完全初心者（ハングルゼロから）
- ハングルの読み書き（母音・子音・パッチム）
- 基本あいさつ・自己紹介フレーズ
- 数字・日付・時間の言い方
- よく使う形容詞・動詞20選

### Level 1 ｜ 初級（基礎の定着）
- 日常会話の基本パターン（〜です、〜ません、〜ましたか？）
- 日記ライティングスタート（3文から）
- カフェ・買い物・交通などシチュエーション別会話
- K-POP・ドラマのフレーズを使った語彙強化

### Level 2 ｜ 中級（応用会話）
- 感情・意見を韓国語で表現する
- 日記を5〜8文に拡張・内容深掘り質疑応答
- テーマ別ディスカッション（旅行・仕事・趣味など）
- 敬語と友達言葉の使い分け

### Level 3 ｜ 上級（実践・流暢さ追求）
- ビジネスシーンの韓国語（メール・電話・プレゼン）
- ニュース・SNS投稿の読解・要約
- 韓国文化・社会トピックでのディスカッション
- 発音・イントネーションの精度アップ

## 6. 授業進行プラン

### マンツーマン授業（1時間）

個人のレベルとペースに完全対応。日記メソッドを中心に、その日の状態に合わせて柔軟に進める。

| 時間 | 内容 | 詳細 |
|------|------|------|
| 0:00〜0:10 | ウォームアップ | 前回の復習・フリートーク（韓国語で） |
| 0:10〜0:30 | 日記読み＆発音矯正 | 生徒が書いた日記を音読 → 発音チェック → 言い換え提案 |
| 0:30〜0:50 | テーマ会話練習 | 日記の内容をきっかけにトピックを深掘り |
| 0:50〜0:55 | 新語彙・表現インプット | その日のテーマに関連した新しい語彙・フレーズ |
| 0:55〜1:00 | まとめ＆次回予告 | 本日の振り返り・次回の日記テーマ提示 |

### グループ授業（1時間）3〜4名

少人数だからこそできる「全員参加型」の授業。お互いの韓国語を聞き合うことで、自然な刺激が生まれる。

| 時間 | 内容 | 詳細 |
|------|------|------|
| 0:00〜0:10 | アイスブレイク | フリートーク・前回復習クイズ |
| 0:10〜0:30 | テーマ学習 | その日のテーマ語彙・フレーズをインプット |
| 0:30〜0:50 | ロールプレイ | 2人ペアで会話練習 → 全体でシェア |
| 0:50〜0:55 | ゲーム・アクティビティ | 韓国語ビンゴ・クイズ・なりきり会話など |
| 0:55〜1:00 | まとめ | 本日の表現まとめ・次回テーマ発表 |
